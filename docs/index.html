OCP Score Criteria

  • Proximity and frequency of nearby transit
  • Ammenity count based on an 800m proximity to one of the following amenities:
    • Grocery store
    • Bank
    • Bar
    • Coffee Shop
    • Library
    • Medical Clinic
    • School

When calculating OCP scores, transit access and amenity counts are weighed 50/50. This isn't meant as a comprehensive survey of Victoria amenities, but rather an illustration of which neighbourhoods are well-resourced and which ones aren't.

See the underlying data and code on Github.
