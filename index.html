Where is it 'suitable' to build an apartment?

Criteria:

  • Only properties within 400m of a bus stop are shown on the map
  • Ammenity count based on an 800m proximity to one of the following amenities:
    • Grocery store
    • Bank
    • Bar
    • Coffee Shop
    • Library
    • Medical Clinic
    • School

This isn't meant as a comprehensive survey of Victoria amenities, but rather an illustration of which neighbourhoods are well-resourced and which ones aren't.

See the underlying data and code on Github.
